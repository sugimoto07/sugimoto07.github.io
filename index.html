Non-Semantic HTML Tags

Tag Name Attributes
<img> HTML Image Tag Defines an image. Self closing. Include src="", alt, width, height
<link> HTML Link Tag Defines the relationship between the current doc and an external resource, usually a style sheet. rel="", href=""
<a> HTML Hyperlink Tag Defines a hyperlink. Includes href="", target="_blank"

Semantic HTML Tags

Tag Name Attributes
<aside> HTML <aside> Element Specifies independent, self-contained content.
<article> HTML <article> Element An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site.
<details> HTML <details> Element Specifies independent, self-contained content.
<figcaption> HTML <figcaption> Element Specifies independent, self-contained content.
<figure> HTML <figure> Element Specifies independent, self-contained content.
<footer> HTML <footer> Element Element defines a footer for a document or section. Typically contains:
  • Authorship information
  • Copyright information
  • Contact information
  • Sitemap
  • Back-to-top links
  • Related documents
<header> HTML <header> Element The <header> element represents a container for introductory content or a set of navigational links.
<main> HTML <main> Element Specifies independent, self-contained content.
<mark> HTML <mark> Element Specifies independent, self-contained content.
<nav> HTML <nav> Element Defines a set of navigation links.
<section> HTML <section> Element According to W3C's HTML documentation: "A section is a thematic grouping of content, typically with a heading."
<summary> HTML <summary> Element Specifies independent, self-contained content.
<time> HTML <time> Element Specifies independent, self-contained content.

CLI Tags

Command Meaning Notes
ls Short listing -l: Long Listing. -a: Include hidden files
cd Change Directory Remember to put in "cd [directory]" and not cd/
pwd Print Working Directory Full path to working directory
mkdir Make Directory Create a new directory
touch Create Empty File The easiest way to create a file, specifically an empty one, is to use the touch command

Git Commands

Tag Name Attributes
git add <directory> Git <add> Command Stage all changes in <directory> for the next commit. Replace <directory> with a <file> to change a specific file.
git clone <https://name-of-the-repository-link> Git <clone> Command This will make a copy of the project to your local workspace so you can start working with it.
git branch <branch-namek> Git <branch> Command This command will create a branch locally.
